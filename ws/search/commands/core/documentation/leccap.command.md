## Lecture Capture

### Lecture Capture (`leccap`)
*Usage:* `leccap <class>`

If `class` is blank, will redirect to [https://leccap.engin.umich.edu/leccap/].

The following classes are supported. Note that case and formatting is ignored when typing a query.

- `eecs280`

---

- `leccap eecs280`
- `leccap EECS 280`
